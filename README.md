# 🤖 A9 Chatbot – Final Group Project

## 📌 Overview
A9 is a chatbot designed to answer questions about our team members.  
It uses a Flask backend and a simple retrieval-based system to provide accurate responses.

---

## 🚀 Features
- 💬 Chat interface with user & bot messages
- ⏳ Typing indicator with realistic delay
- 🧠 Retrieval-based response system
- ⚠️ Handles unknown and empty inputs
- 📱 Responsive design (mobile + desktop)
- 🎭 Chatbot personality (A9 Assistant)

---

## 🏗️ Architecture
User Input → Frontend (HTML/CSS/JS)  
→ Flask Backend (`app.py`)  
→ Dataset Matching (keyword retrieval)  
→ Response returned to UI
